# Planner Minimal

Минималистичный premium планёр задач, заметок и календаря. Pinterest aesthetic. PWA — устанавливается на iPhone через Safari.

```bash
npm install
npm run dev
npm run build
```

## Установка как PWA на iPhone

1. Открой сайт в **Safari**
2. Нажми кнопку «Поделить» → «На экран «Домой»»
3. Готово — приложение установлено на телефоне

## Стек

- **Next.js 15** — App Router
- **TypeScript** — строгая типизация
- **TailwindCSS** — стили
- **Framer Motion** — анимации
- **LocalStorage** — хранение данных
- **PWA** — устанавливается на iPhone

## Функции

| Функция | Описание |
|---|---|
| Onboarding | 4 слайда при первом запуске |
| Главный экран | Приветствие, цитата дня, прогресс, задачи |
| Задачи | Создание, выполнение, удаление свайпом |
| Автоперенос | Невыполненные задачи → следующий день |
| Приоритеты | Обычный / Важно / Срочно |
| Календарь | Кастомная сетка с индикаторами задач |
| Заметки | Цветные карточки в две колонки |
| Dark mode | Тёплые тёмные тона |
| Фильтры | Все / Активные / Выполненные |
